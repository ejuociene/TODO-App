# TODO App

## Overview and Features

Sample ToDo app with features:
- Add new task
- Edit tasks
- Filter tasks by category
- Mark tasks as Done
- View and remove Completed tasks
- Edit categories 
- Toggle Light and Dark Mode
- Responsive layout

### Built with
- React.js
- Semantic HTML5 markup
- Flexbox
- Saving data on Local Storage
